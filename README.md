# Swiss Nursing Home Management API

Dieses Repository beinhaltet die API-Definition für eine umfassende Management-Lösung für eine Schweizer Pflegeheimgruppe.

## Zusammenfassung

Das System unterstützt über 500 Mitarbeitende und deckt folgende Kernbereiche ab:

*   **HR (Personalwesen):** Verwaltung von Mitarbeitenden, Zeiterfassung und Absenzenmanagement.
*   **Einsatzplanung (Scheduling):** Dienstplanung unter Berücksichtigung von GAV-Vorgaben (Gesamtarbeitsvertrag).
*   **Bewohnerverwaltung (Residents):** Administration der Heiminsassen inklusive Stammdaten.
*   **Pflegeplanung (Care Planning):** Erstellung und Verwaltung von Pflegeplänen, Zielen und Interventionen.
*   **Pflegedurchführung (Care Execution):** Dokumentation täglicher Pflegemassnahmen und Vitalwerte.
*   **Abrechnung (Billing):** Fakturierung und Integration mit Krankenkassen (Tarmed, BESA).
