# Systemlandschaft

Basierend auf den bereitgestellten Informationen (Stand 01.12.2025).

## Übersicht der Systeme

| System | Hersteller | Funktionalität |
| :--- | :--- | :--- |
| **Pep** | Polypoint | Einsatzplanung und Präsenzerfassung |
| **Nexus** | Nexus | Hauptsystem (Finanzen, Lohnabrechnung, Bewohneradministration) |
| **EasyDOK** | EasyDOK | Pflege- und Leistungsplanung |
| **Rai LTCF** | BESA QSys | Bedarfserfassungsinstrument zur Berechnung der Pflegestufe |

## Datenfluss und Integration

*   **Stammdatenhoheit:** Die Bewohnerstammdaten werden in **Nexus** geführt.
*   **Datenverteilung:** Die Stammdaten werden von Nexus an **EasyDOK** und **Rai LTCF** verteilt.
*   **Lücken:** Es existiert **keine Schnittstelle** zwischen Nexus und PEP.
